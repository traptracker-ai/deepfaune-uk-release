To use your own logo on the dashboard, drop an image file here named:
  logo.png   (or logo.svg / logo.jpg / logo.webp)

It will appear in the dashboard header automatically (no rebuild needed if the
static folder is mounted; otherwise rebuild the dashboard image). Recommended:
a horizontal logo around 200-400px wide, or a square mark ~120px.

If no logo file is present, the dashboard shows the built-in TRAP TRACKER wordmark.
